# sahil123
FILE HANDLING SYSTEM (PYTHON)

This is a simple beginner-friendly Python project. It helps you create, read, and update files using basic file handling concepts and the pathlib module.

FEATURES

* Create a new file
* Read an existing file
* Update file:

  * Rename file
  * Overwrite content
  * Append data
* Display all files and folders

TECHNOLOGIES USED

* Python 3
* pathlib module

HOW IT WORKS

When you run the program, you will see:

press 1 for creating a file
press 2 for reading a file
press 3 for updating a file

1. CREATE FILE

* Enter file name
* Enter data
* File will be created if it does not exist

2. READ FILE

* Enter file name
* Program will display file content

3. UPDATE FILE

* Rename file
* Overwrite data
* Append data

2. Go to folder:
   cd your-repo-name
